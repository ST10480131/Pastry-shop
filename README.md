# Sweet cravings Pastry-shop
Overview

Sweet Craving Pastry Shop is a fully designed front-end website built using HTML5, CSS3, and JavaScript.
The site includes a responsive navigation menu with a hamburger toggle, smooth animations, accessible focus states, and clean layouts across all pages.

This project simulates a real bakery website, featuring:

 Homepage with specials

 About Us page

 Order Page

 Contact Page with embedded Google Maps

 Pastry Mood Picker with filtering, product cards, and a working cart system
 
#Page	Description

Home page	Welcoming hero section + current specials
About Us	History, mission, and vision of the bakery
Order Now	Product ordering section (if included)
Contact Us	Contact details + embedded Google Map
Pastry Mood Picker	Category-based pastry suggestions + cart
